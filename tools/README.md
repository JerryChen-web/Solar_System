# Tools

This folder is reserved for future data pipeline scripts.

V0.1 intentionally does not include executable NASA/JPL fetch scripts. Planned V3 tools include:

- `fetch_horizons_vectors.py`
- `fetch_horizons_elements.py`
- `convert_horizons_to_json.py`
- `validate_orbit_error.py`

Future scripts must write raw data under `data/raw` and processed app-ready data under `data/processed`.
