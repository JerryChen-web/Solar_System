# Tests

Automated tests use Vitest for astronomy, data, and UI helper coverage. V0.8 includes coverage for fixture source switching, local import selection, fallback behavior, active source display, reset to default, and report export helpers.

Recommended validation commands:

- `npm.cmd run typecheck`
- `npm.cmd run test`
- `npm.cmd run build`
- Manual browser smoke test through the Vite dev server.
