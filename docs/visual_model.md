# Visual Model

V0.1 separates physical values from display values.

- Real radius remains in meters.
- Real distance remains in meters or AU-derived meters.
- Scene positions use Three.js units generated by a visual scale function.
- Planet radii use per-body visual scale plus min/max clamps.

V1.0 uses procedural runtime display enhancements: generated planet textures, glow shells, atmosphere shells, ring bands, starfield points, and deterministic small-body belt points. These visuals do not overwrite physical values and do not download textures.
