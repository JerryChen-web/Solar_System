# Visual Model

V0.1 separates physical values from display values.

- Real radius remains in meters.
- Real distance remains in meters or AU-derived meters.
- Scene positions use Three.js units generated by a visual scale function.
- Planet radii use per-body visual scale plus min/max clamps.

The MVP uses colored placeholder materials, simple orbit lines, labels, basic lighting, and a lightweight Saturn ring mesh. It does not download textures.
