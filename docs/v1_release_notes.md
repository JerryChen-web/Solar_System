# Solar_System V1.0.0 Release Notes

V1.0.0 is the first public stable visual release for Solar_System.

## Highlights

- Public GitHub Pages release remains available at `https://JerryChen-web.github.io/Solar_System/`.
- Visible app label and browser title now read `Solar_System V1.0.0`.
- NASA-like procedural rendering pass with stronger Sun glow, deeper starfield, improved planet identities, and cleaner orbit presentation.
- Deterministic asteroid belt and sparse Kuiper belt visualization.
- Planet ecosystem focus mode for all eight major planets.
- Click a supported planet to focus; press `ESC` or `Back to solar system` to return.

## Preserved From V0.9

- GitHub Pages base path `/Solar_System/`.
- Deployment workflow and static Vite build behavior.
- Kepler animation, time scale controls, date jump, object selection, Follow / Stop Follow, Debug panel, Validation Dashboard, Position Table, Moon-Earth check, report exports, Reference Import panel, fixture switching, and fallback reset.

## Limits

V1.0.0 does not add live NASA/JPL Horizons, SPICE, backend services, databases, real-time ephemerides, spacecraft trajectories, large texture packs, or external runtime asset loading.
